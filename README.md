# Spike Forms Agent Skills

AI agent skills for integrating with [Spike Forms](https://spike.ac) — a form backend that handles submissions, email notifications, webhooks, and file uploads. No backend code needed.

## Install

```bash
npx skills add spike-forms/agent-skills
```

Or via npm:

```bash
npm install @spike-forms/agent-skills
npx skills-npm
```

## Skills

- **spike-forms** — Complete integration guide: HTML forms, React, file uploads, webhooks, special fields, API reference.
